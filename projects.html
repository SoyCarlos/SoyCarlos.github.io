What Are the Odds? - Python

What are the odds is a game I learned in highschool. It's very similar to a dare in that it is usually used when a group of friends is trying to get one another to do something they usually wouldn't. The rules are simple, the friend being challenged names the odds, 1 to whatever they please. Obviously the higher the bound the lower the chance they'll have to do it. Once it's named, say 1 to 10. The challenger and the challenged countdown from 3 and say a number between 1 and, in this case, 10, at the same time. If they say the same number or the two numbers add up to the upper bound, the challenged friend has to do whatever they agreed upon.

This short program is the first project I created upon having learned to code in Python. This is a short, but fun, game run within terminal.
